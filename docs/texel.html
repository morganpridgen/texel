Texel Documentation - TEXEL


bool TXL_Init()

Sets up modules Texel relies on. Returns 1 on success, 0 on failure.
int main() { if (!TXL_Init()) return 1; // other code }

bool TXL_Events(TXL_Display*)

Handles display events. Argument is your display. Returns 0 when a shutdown signal has been sent to the application, 1 otherwise.
TXL_Display display; // other code while (loop) { if (!TXL_Events(&display)) loop = 0; // other code }

void TXL_End()

Shuts down all modules Texel relies on.
int main() { // other code TXL_End(); }
