Texel Documentation - Textures


Class

TXL_Texture

Class for texture rendering.

bool TXL_Texture::loaded()

Returns 1 if the texture is loaded, 0 otherwise.
TXL_Texture tex; int x, y; // other code if (tex.loaded()) tex.render(x, y); else TXL_RenderQuad({x, y, 16, 16}, {1, 1, 1, 1});

bool TXL_Texture::load(const char path[], int w, int h), bool TXL_Texture::load(const char path[])

Loads path. If both w and h are not -1, then all rendering functions will assume the image's resolution is w*h. If either is -1, then the image's actual resolution is used instead.
Returns 1 on success, 0 otherwise.
If only path is given, w and h are assumed to be -1. TXL_Texture tex; if (!tex.load(TXL_DataPath("player.png"), 16, 16)) return 0;

void TXL_Texture::free()

Deletes the texture. TXL_Texture tex; // other code tex.free();

void TXL_Texture::render(float x, float y, float sX, float sY, float r), void TXL_Texture::render(float x, float y, float sX, float sY), void TXL_Texture::render(float x, float y, float r), void TXL_Texture::render(float x, float y)

Draws the texture centered at (x, y). The texture's width is multiplied by sX, and the height by sY. The texture will be rotated by r radians.
If sX and sY are not given, they are assumed to be 1. r will be 0 if not given. TXL_Texture tex; // other code tex.render(128, 128, 2, 2, 3.14 / 6.0f);

void TXL_Texture::setClip(float cL, float cR, float cT, float cB)

Set the portion of the texture that gets rendered. cL and cR are the left and right edges of the portion of the texture to be rendered, while cT and cB are the top and bottom.
This also changes the size of the rendered texture.
TXL_Texture tex; if (!tex.load(TXL_DataPath("player.png"), 32, 16)) return 0; // other code tex.setClip(0, 16, 0, 16); tex.render(128, 128);

void TXL_Texture::setColorMod(float r, float g, float b, float a), void TXL_Texture::setColorMod(float r, float g, float b), void TXL_Texture::setColorMod(float a)

Sets the color modulation for the texture.
Every color value for every pixel will be multiplied by the value given here.
If the RGB only or the alpha only versions are used, the other color channels remain unchanged.
TXL_Texture tex; if (!tex.load(TXL_DataPath("player.png"), 16, 16)) return 0; // other code tex.setColorMod(0, 1, 0, 0.5f); tex.render(128, 128);
