Texel Documentation - Display


Constants

TXL_IntResX = 640.0f, TXL_IntResY = 360.0f

The size of the display in screen coordinates. Rendering functions will handle upscaling to higher resolutions.

Class

TXL_Display

Class representing a display window.

bool TXL_Display::init(const char name[])

Sets up the display window. The window title will include name.
Returns 1 on success, 0 otherwise.
TXL_Display display; if (!display.init("Example Window")) return 0;

void TXL_Display::end()

Closes the display window. TXL_Display display; if (!display.init("Example Window")) return 0; // other code display.end();

void TXL_Display::refresh()

All rendering function calls don't actually modify the screen, they affect a buffer. This function copies the buffer to the screen to make everything visible.
In addition, the buffer will be filled with the color specified by the previous call to TXL_Display::setFill.
TXL_Display display; // other code TXL_RenderQuad({160, 90, 320, 180}, {1, 1, 1, 1}); display.refresh();

void TXL_Display::setFill(float r, float g, float b)

Sets the color the screen is filled with when TXL_Display::refresh is called.
TXL_Display display; display.setFill(1, 1, 1); // other code display.refresh();
